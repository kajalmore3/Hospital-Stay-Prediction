# Hospital Stay Prediction

## Project Overview
This project predicts the length of stay of patients in hospitals using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- XGBoost

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Best Model
Random Forest / XGBoost

## Project Workflow
1. Data Collection
2. Data Cleaning
3. EDA
4. Feature Engineering
5. Model Building
6. Evaluation
7. Hyperparameter tuning

## Results
The Random Forest model achieved the best accuracy in predicting patient stay duration.
